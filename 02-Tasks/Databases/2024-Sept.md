# Задача 1:
```sql
SELECT AVG(price)
FROM Laptop
LEFT JOIN Product ON Laptop.model = Product.model
WHERE ram BETWEEN 16 AND 32 OR maker IS NULL
```

# Задача 2:
Отговор: Б
```sql
SELECT maker, COUNT(DISTINCT Laptop.model)
FROM Laptop
RIGHT JOIN Product
ON Laptop.model = Product.model
AND screen = 14
GROUP BY maker;
```
