Нека е дефиниран масивът int arr[] = { 1, 2, 3 };
Срещу всеки от изразите да се посочи каква ще бъде неговата оценка
- arr[1] == *(arr+2) _________________
- arr == &arr[0] _________________
- (arr+1) == &arr[1] _________________
- *arr == arr[0] _________________

Отговор:
- arr[1] == *(arr+2) false
- arr == &arr[0] true
- (arr+1) == &arr[1] true
- *arr == arr[0] true
