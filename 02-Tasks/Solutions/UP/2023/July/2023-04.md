Да се посочи какво ще изведе на стандартния изход следният фрагмент.
```c++
int x = 2;
int arr[] = {10, 20, 30};
cout << "\nA: " << 5./x;
cout << "\nB: " << (x << 4);
cout << "\nC: " << arr[!x];
cout << "\nD: " << *(arr+x);
cout << "\nE: " << (2 + x++);
```

Отговор:
```
A: A: 2.5
B: 32
C: 10
D: 30
E: 4
```
